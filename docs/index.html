项目定义

你现在真正要做的事

稳定主线

把 Autoware + CARLA 的稳定闭环先跑通，优先服务规划控制验证，而不是一开始就做重型数字孪生。

自动化验证

把启动、运行、批量回归、回放、报告和 KPI 门禁固化，形成可重复使用的实验工作流。

现场场景前移

用现场地图、点云和问题案例构建 site proxy 与 corner case，让现场问题能提前在仿真里复现。

未来算法接入

为 UE5 和端到端算法准备实验底座，但不让它们反向拖慢当前闭环主线。
