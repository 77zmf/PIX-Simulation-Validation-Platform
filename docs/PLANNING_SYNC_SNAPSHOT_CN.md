# 规划同步快照

## 当前统一结论

- 本季度主线只按 `Autoware Universe + CARLA 0.9.15 + UE4.26` 推进
- 不再保留第二套独立仿真 runtime 作为当前执行路线或研究路线
- `BEVFusion` 是主感知基线
- `UniAD-style` 和 `VADv2` 只作为 `E2E shadow` 研究项存在

## 为什么这样定

- 这样能让稳定闭环、自动化、资产束和研究线共享同一套运行时
- 可以避免多条 runtime 基线带来的重复维护和结论失真
- 能把所有对比结论都收口到同一套 `CARLA 0.9.15` 场景和 KPI

## 当前执行口径

- 主仿真运行环境：公司 Ubuntu 主机
- 主仿真版本：`CARLA 0.9.15`
- 主引擎口径：`UE4.26`
- 研究线运行方式：全部挂到 `stable` 栈
